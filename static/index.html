Easy Show Downloader

    The server will check for new episodes each hour. You can also click the button above to do it manually.

    Enter regular expressions in the box to the left. Your regex should have two named capture groups, name, which captures the name of the show, and episode which captures the episode number.

    Here's an example:

    ^\[SubsPlease] (?<name>Akudama Drive) - (?<episode>\d+) \(1080p\).*$

    This will download files with names like

    [SubsPlease] Akudama Drive - 10 (1080p) [7F2BE509].mkv
